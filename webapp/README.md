# Policy Preference Matcher - Frontend

React.js frontend application for matching policy preferences with elections and candidates.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- **Sidebar with Basic Information Entry:**
  - Age input
  - Gender selection dropdown
  - City of residence input
  - Income slider (0 - $1M+)
  - Donation amount input

- **Main Content Area:**
  - Placeholder for search results (elections and candidates)

## Tech Stack

- React 18
- Vite (build tool)
- Modern CSS with clean, responsive design
